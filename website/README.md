# Gas Town Documentation

This repository contains the Docusaurus-powered documentation site for
[Gas Town](https://github.com/steveyegge/gastown).

## Installation

```bash
npm install
```

## Local development

```bash
npm run start
```

The development server runs at http://localhost:3000/.

## Build

```bash
npm run build
```

The static site output is generated in the `build` directory.

## Preview the production build

```bash
npm run serve
```

## Deployment

```bash
npm run deploy
```

Configure `CLOUDFLARE_API_TOKEN` and `CLOUDFLARE_ACCOUNT_ID` as needed for Cloudflare Pages deployments.
