## gt synthesis status

Show synthesis readiness

### Synopsis

Show whether a convoy is ready for synthesis.

Displays:
  - Convoy metadata
  - Leg completion status
  - Available leg outputs
  - Formula synthesis configuration

```
gt synthesis status <convoy-id> [flags]
```

### Options

```
  -h, --help   help for status
```
