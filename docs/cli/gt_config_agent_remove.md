## gt config agent remove

Remove custom agent

### Synopsis

Remove a custom agent definition from town settings.

This removes a custom agent from your town settings. Built-in agents
(claude, gemini, codex) cannot be removed.

Examples:
  gt config agent remove claude-glm

```
gt config agent remove <name> [flags]
```

### Options

```

```
