## gt theme apply

Apply theme to running sessions

### Synopsis

Apply theme to running Gas Town sessions.

By default, only applies to sessions in the current rig.
Use --all to apply to sessions across all rigs.

```
gt theme apply [flags]
```

### Options

```
  -a, --all    Apply to all rigs, not just current
  -h, --help   help for apply
```
