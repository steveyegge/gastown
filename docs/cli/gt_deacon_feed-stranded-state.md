## gt deacon feed-stranded-state

Show feed-stranded state for tracked convoys

### Synopsis

Display the current feed-stranded tracking state including:
- Feed counts per convoy
- Cooldown status
- Last feed times

This helps the Deacon understand which convoys have been recently fed.

```
gt deacon feed-stranded-state [flags]
```
