## gt mail channel show

Show channel messages

### Synopsis

Show messages from a broadcast channel.

Displays the channel's retention policy and all messages sorted by
creation time, including sender, timestamp, and a body preview.
Use --json for machine-readable output.

```
gt mail channel show <name> [flags]
```

### Options

```
  -h, --help   help for show
      --json   Output as JSON
```
