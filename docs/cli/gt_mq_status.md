## gt mq status

Show detailed merge request status

### Synopsis

Display detailed information about a merge request.

Shows all MR fields, current status with timestamps, dependencies,
blockers, and processing history.

Example:
  gt mq status gp-mr-abc123

```
gt mq status <id> [flags]
```

### Options

```
  -h, --help   help for status
      --json   Output as JSON
```
