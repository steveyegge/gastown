## gt town next

Switch to next town session (mayor/deacon)

### Synopsis

Switch to the next town-level session in the cycle order.
Town sessions cycle between Mayor and Deacon.

This command is typically invoked via the C-b n keybinding when in a
town-level session (Mayor or Deacon).

```
gt town next [flags]
```

### Options

```
  -h, --help             help for next
      --session string   Override current session (used by tmux binding)
```
