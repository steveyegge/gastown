## gt mayor attach

Attach to the Mayor session

### Synopsis

Attach to the running Mayor tmux session.

Attaches the current terminal to the Mayor's tmux session.
Detach with Ctrl-B D.

```
gt mayor attach [flags]
```

### Options

```
      --agent string   Agent alias to run the Mayor with (overrides town default)
  -h, --help           help for attach
```
