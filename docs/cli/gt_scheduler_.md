## gt scheduler 

Resume scheduler dispatch

### Synopsis

Resume the dispatch scheduler after a pause.

Allows the scheduler to resume dispatching work to polecats.

Examples:
  gt scheduler resume     # Resume dispatch

```
gt scheduler  [flags]
```
