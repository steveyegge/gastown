## gt convoy status

Show convoy status

### Synopsis

Show detailed status for a convoy.

Displays convoy metadata, tracked issues, and completion progress.
Without an ID, shows status of all active convoys.

```
gt convoy status [convoy-id] [flags]
```

### Options

```
  -h, --help   help for status
      --json   Output as JSON
```
