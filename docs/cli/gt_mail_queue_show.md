## gt mail queue show

Show queue details

### Synopsis

Show details about a mail queue.

Displays the queue's claim pattern, status, and message counts.

Examples:
  gt mail queue show work
  gt mail queue show dispatch --json

```
gt mail queue show <name> [flags]
```

### Options

```
  -h, --help   help for show
      --json   Output as JSON
```
