## gt mail queue delete

Delete a queue

### Synopsis

Delete a mail queue.

This permanently removes the queue bead. Messages in the queue are not affected.

Examples:
  gt mail queue delete work

```
gt mail queue delete <name> [flags]
```

### Options

```
  -h, --help   help for delete
```
