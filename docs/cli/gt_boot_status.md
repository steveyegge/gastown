## gt boot status

Show Boot status

### Synopsis

Show Boot's current status and last execution.

Displays:
  - Whether Boot is currently running
  - Last action taken (start/wake/nudge/nothing)
  - Timing information
  - Degraded mode status

```
gt boot status [flags]
```

### Options

```
  -h, --help   help for status
      --json   Output as JSON
```
