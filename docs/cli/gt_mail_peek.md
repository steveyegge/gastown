## gt mail peek

Show preview of first unread message

### Synopsis

Display a compact preview of the first unread message.

Useful for status bar popups - shows subject, sender, and body preview.
Exits silently with code 1 if no unread messages.

```
gt mail peek [flags]
```

### Options

```
  -h, --help   help for peek
```
