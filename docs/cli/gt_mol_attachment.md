## gt mol attachment

Show attachment status of a pinned bead

### Synopsis

Show which molecule is attached to a pinned bead.

Example:
  gt molecule attachment gt-abc

```
gt mol attachment <pinned-bead-id> [flags]
```

### Options

```
  -h, --help   help for attachment
      --json   Output as JSON
```
