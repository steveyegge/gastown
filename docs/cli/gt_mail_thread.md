## gt mail thread

View a message thread

### Synopsis

View all messages in a conversation thread.

Shows messages in chronological order (oldest first).

Examples:
  gt mail thread thread-abc123

```
gt mail thread <thread-id> [flags]
```

### Options

```
  -h, --help   help for thread
      --json   Output as JSON
```
