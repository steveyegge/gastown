## gt krc config reset

Reset TTL configuration to defaults

### Synopsis

Reset the KRC TTL configuration file to built-in defaults.

This overwrites any custom TTL patterns and restores the default
prune interval, minimum retain count, and per-type TTLs.

```
gt krc config reset [flags]
```

### Options

```
  -h, --help   help for reset
```
