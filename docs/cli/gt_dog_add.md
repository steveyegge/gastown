## gt dog add

Create a new dog in the kennel

### Synopsis

Create a new dog in the kennel with multi-rig worktrees.

Each dog gets a worktree per configured rig (e.g., gastown, beads).
The dog starts in idle state, ready to receive work from the Deacon.

Example:
  gt dog add alpha
  gt dog add bravo

```
gt dog add <name> [flags]
```
