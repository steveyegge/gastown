## gt mayor start

Start the Mayor session

### Synopsis

Start the Mayor tmux session.

Creates a new detached tmux session for the Mayor and launches Claude.
The session runs in the workspace root directory.

```
gt mayor start [flags]
```

### Options

```
      --agent string   Agent alias to run the Mayor with (overrides town default)
  -h, --help           help for start
```
