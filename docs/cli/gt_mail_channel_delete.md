## gt mail channel delete

Delete a channel

### Synopsis

Delete a broadcast channel and its configuration.

This removes the channel bead. Existing messages that were broadcast
to the channel are not deleted but will no longer be queryable by
channel name.

```
gt mail channel delete <name> [flags]
```

### Options

