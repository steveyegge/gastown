## gt deacon start

Start the Deacon session

### Synopsis

Start the Deacon tmux session.

Creates a new detached tmux session for the Deacon and launches Claude.
The session runs in the workspace root directory.

```
gt deacon start [flags]
```

### Options

```
      --agent string   Agent alias to run the Deacon with (overrides town default)
  -h, --help           help for start
```
