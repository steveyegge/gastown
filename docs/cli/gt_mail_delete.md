## gt mail delete

Delete messages

### Synopsis

Delete (acknowledge) one or more messages.

This closes the messages in beads.

Examples:
  gt mail delete hq-abc123
  gt mail delete hq-abc123 hq-def456 hq-ghi789

```
gt mail delete <message-id> [message-id...] [flags]
```
