## gt mail channel list

List all channels

### Synopsis

List all broadcast channels in the workspace.

Displays each channel's name, retention policy, status, and creator.
Use --json for machine-readable output.

```
gt mail channel list [flags]
```

### Options

```
  -h, --help   help for list
      --json   Output as JSON
```
