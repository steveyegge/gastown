## gt wl sync

Pull upstream changes into local wl-commons fork

### Synopsis

Sync your local wl-commons fork with the upstream hop/wl-commons.

If you have a local fork of wl-commons (created by gt wl join), this pulls
the latest changes from upstream.

EXAMPLES:
  gt wl sync                # Pull upstream changes
  gt wl sync --dry-run      # Show what would change

```
gt wl sync [flags]
```

### Options

```
      --dry-run   Show what would change without pulling
  -h, --help      help for sync
```
