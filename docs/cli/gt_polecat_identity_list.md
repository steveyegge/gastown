## gt polecat identity list

List polecat identity beads in a rig

### Synopsis

List all polecat identity beads in a rig.

Shows:
  - Polecat name
  - Agent state
  - Current hook (if any)
  - Whether worktree exists

Example:
  gt polecat identity list gastown
  gt polecat identity list gastown --json

```
gt polecat identity list <rig> [flags]
```

### Options

```
  -h, --help   help for list
      --json   Output as JSON
```
