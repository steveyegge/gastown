## gt krc stats

Show statistics about ephemeral data

### Synopsis

Display statistics about events and their TTL status.

Shows:
  - File sizes and event counts
  - Events by type and age
  - TTL configuration and expiration status

```
gt krc stats [flags]
```

### Options

```
  -h, --help   help for stats
      --json   Output in JSON format
```
