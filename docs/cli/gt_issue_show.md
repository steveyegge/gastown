## gt issue show

Show the current issue

### Synopsis

Show the current issue ID from the tmux session environment.

Displays the issue ID currently set for the tmux status line, or
indicates that no issue is set.

```
gt issue show [flags]
```

### Options

```
  -h, --help   help for show
```
