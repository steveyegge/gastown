## gt namepool themes

List available themes and their names

### Synopsis

List available namepool themes or show names in a specific theme.

Without arguments, lists all themes with a preview of their names.
With a theme name argument, shows all names in that theme.

```
gt namepool themes [theme] [flags]
```

### Options

```
  -h, --help   help for themes
```
