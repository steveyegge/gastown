## gt session status

Show session status details

### Synopsis

Show detailed status for a polecat session.

Displays running state, uptime, session info, and activity.

```
gt session status <rig>/<polecat> [flags]
```

### Options

```
  -h, --help   help for status
      --json   Output as JSON
```
