## gt session list

List all sessions

### Synopsis

List all running polecat sessions.

Shows session status, rig, and polecat name. Use --rig to filter by rig.

```
gt session list [flags]
```

### Options

```
  -h, --help         help for list
      --json         Output as JSON
      --rig string   Filter by rig name
```
