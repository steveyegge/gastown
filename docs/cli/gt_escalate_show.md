## gt escalate show

Show details of an escalation

### Synopsis

Display detailed information about an escalation.

Examples:
  gt escalate show hq-abc123
  gt escalate show hq-abc123 --json

```
gt escalate show <escalation-id> [flags]
```

### Options

```
  -h, --help   help for show
      --json   Output as JSON
```
