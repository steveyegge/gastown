## gt namepool add

Add a custom name to the pool

### Synopsis

Add a custom name to the rig's polecat name pool.

The name is appended to the pool and saved in the rig settings.
Duplicate names are silently ignored.

```
gt namepool add <name> [flags]
```

### Options

```

```
