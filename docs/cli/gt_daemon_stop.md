## gt daemon stop

Stop the daemon

### Synopsis

Stop the running Gas Town daemon.

Sends a stop signal to the daemon process and waits for it to exit.
The daemon must be running or this command returns an error.

Examples:
  gt daemon stop

```
gt daemon stop [flags]
```
