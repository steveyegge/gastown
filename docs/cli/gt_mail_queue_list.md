## gt mail queue list

List all queues

### Synopsis

List all beads-native mail queues.

Shows queue names, claim patterns, and status.

Examples:
  gt mail queue list
  gt mail queue list --json

```
gt mail queue list [flags]
```

### Options

```
  -h, --help   help for list
      --json   Output as JSON
```
