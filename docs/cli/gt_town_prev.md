## gt town prev

Switch to previous town session (mayor/deacon)

### Synopsis

Switch to the previous town-level session in the cycle order.
Town sessions cycle between Mayor and Deacon.

This command is typically invoked via the C-b p keybinding when in a
town-level session (Mayor or Deacon).

```
gt town prev [flags]
```

### Options

```
  -h, --help             help for prev
      --session string   Override current session (used by tmux binding)
```
