## gt dolt sql

Open Dolt SQL shell

### Synopsis

Open an interactive SQL shell to the Dolt database.

Works in both embedded mode (no server) and server mode.
For multi-client access, start the server first with 'gt dolt start'.

```
gt dolt sql [flags]
```
