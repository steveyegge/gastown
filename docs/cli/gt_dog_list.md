## gt dog list

List all dogs in the kennel

### Synopsis

List all dogs in the kennel with their status.

Shows each dog's state (idle/working), current work assignment,
and last active timestamp.

Examples:
  gt dog list
  gt dog list --json

```
gt dog list [flags]
```

### Options

```
  -h, --help   help for list
      --json   Output as JSON
```
