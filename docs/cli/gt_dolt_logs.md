## gt dolt logs

View Dolt server logs

### Synopsis

View the Dolt server log file.

```
gt dolt logs [flags]
```

### Options

```
  -f, --follow      Follow log output
  -h, --help        help for logs
  -n, --lines int   Number of lines to show (default 50)
```
