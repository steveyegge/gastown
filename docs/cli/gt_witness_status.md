## gt witness status

Show witness status

### Synopsis

Show the status of a rig's Witness.

Displays running state, monitored polecats, and statistics.

```
gt witness status <rig> [flags]
```

### Options

```
  -h, --help   help for status
      --json   Output as JSON
```
