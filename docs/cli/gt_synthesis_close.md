## gt synthesis close

Close convoy after synthesis

### Synopsis

Close a convoy after synthesis is complete.

This marks the convoy as complete and triggers any configured notifications.

```
gt synthesis close <convoy-id> [flags]
```

### Options

```
  -h, --help   help for close
```
