## gt namepool set

Set the namepool theme for this rig

### Synopsis

Set the namepool theme used for naming new polecats in this rig.

Changes the theme and saves it to the rig settings. Existing polecat
names are not affected. Use 'gt namepool themes' to see available themes.

```
gt namepool set <theme> [flags]
```

### Options

```
  -h, --help   help for set
```
