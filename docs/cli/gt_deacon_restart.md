## gt deacon restart

Restart the Deacon session

### Synopsis

Restart the Deacon tmux session.

Stops the current session (if running) and starts a fresh one.

```
gt deacon restart [flags]
```

### Options

```
      --agent string   Agent alias to run the Deacon with (overrides town default)
  -h, --help           help for restart
```
