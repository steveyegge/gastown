## gt namepool reset

Reset the pool state (release all names)

### Synopsis

Reset the polecat name pool, releasing all claimed names.

All names become available for reuse. This does not change the theme
or remove custom names from the configuration.

```
gt namepool reset [flags]
```

### Options

```
  -h, --help   help for reset
```
