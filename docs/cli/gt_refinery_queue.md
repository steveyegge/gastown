## gt refinery queue

Show merge queue

### Synopsis

Show the merge queue for a rig.

Lists all pending merge requests waiting to be processed.
If rig is not specified, infers it from the current directory.

```
gt refinery queue [rig] [flags]
```

### Options

```
  -h, --help   help for queue
      --json   Output as JSON
```
