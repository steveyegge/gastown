## gt version

Print version information

### Synopsis

Print the gt version, build type, git branch, and commit hash.

Output includes the semantic version, whether this is a dev or release build,
and the git revision the binary was built from (if available).

```
gt version [flags]
```

### Options

```
  -h, --help      help for version
      --short     Output only the version number (e.g., 0.5.0-362)
  -v, --verbose   Show extended version info including timestamp
```
