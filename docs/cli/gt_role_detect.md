## gt role detect

Force cwd-based role detection (debugging)

### Synopsis

Detect role from current working directory, ignoring GT_ROLE env var.

This is useful for debugging role detection issues.

```
gt role detect [flags]
```

### Options

```
  -h, --help   help for detect
```
