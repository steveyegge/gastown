## gt krc auto-prune-status

Show auto-prune scheduling state

### Synopsis

Display the auto-prune scheduling state including:
  - Last prune time and result
  - Total prunes and bytes freed
  - Time until next scheduled prune

```
gt krc auto-prune-status [flags]
```

### Options

```
  -h, --help   help for auto-prune-status
```
