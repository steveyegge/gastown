## gt tap list

List available tap handlers

### Synopsis

List all tap handlers (guards, audits, injectors, checks).

Shows both registered (from registry.toml) and built-in tap commands.

Examples:
  gt tap list               # Show all available handlers
  gt tap list --guards      # Show only guard handlers

```
gt tap list [flags]
```

### Options

```
      --guards   Show only guard handlers
  -h, --help     help for list
```
