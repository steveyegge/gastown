## gt warrant list

List pending warrants

### Synopsis

List all pending (unexecuted) warrants.

Use --all to include executed warrants.

Examples:
  gt warrant list
  gt warrant list --all

```
gt warrant list [flags]
```

### Options

```
  -a, --all    Include executed warrants
  -h, --help   help for list
```
