## gt mol detach

Detach molecule from a pinned bead

### Synopsis

Remove molecule attachment from a pinned/handoff bead.

This clears the attached_molecule and attached_at fields from the bead.

Example:
  gt molecule detach gt-abc

```
gt mol detach <pinned-bead-id> [flags]
```
