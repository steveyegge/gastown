## gt issue clear

Clear the current issue from status line

### Synopsis

Clear the current issue from the tmux session environment.

Removes the issue ID so it no longer appears in the tmux status line.

```
gt issue clear [flags]
```

### Options

```
  -h, --help   help for clear
```
